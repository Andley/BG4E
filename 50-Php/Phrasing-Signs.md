#### 圖析符號

| 符號  | 含意 |
| :---: | ---- |
| **粗體字** | 希臘文動詞 |
| *斜體字* | 希臘文分詞、不定詞 |
| ¬ | OGNTa 換行標記 |
| ¶ | OGNTa 段落標記 |
| S/P/C/A | 主要 & 從屬子句的元素用大寫 S/P/C/A |
| { s/p/c/a } | 內嵌子句的元素用小寫 s/p/c/a) |
| ...... | 省略、改換位置之前的字詞 or 片語 |
| « » | 改換位置之後的字詞 or 片語 |
| ( ) | 片語 |
| ‹ › | 片語中的片語 |
| ◖、◗ | 片語的前半段、後半段|
| °¹、°² etc.  | 指向、修飾其他的字詞 or 片語 |
| <mark>°¹</mark>、<mark>°²</mark> etc. | 被指向、被修飾的字詞 or 片語 |
| 🕍︎、🕇、🕊️ | 聖父、聖子、聖靈 |
| ↴ |  指向下文的字詞 or 片語 |
| ——————— | 分隔線 |
